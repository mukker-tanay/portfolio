# Portfolio Project

## Tech Stack

### Backend
- **Framework**: Django + Django REST Framework
- **Database**: PostgreSQL
- **Auth**: Admin-only
- **Hosting**: Self-hosted (Gunicorn/Uvicorn)

### Frontend
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Motion**: Framer Motion
- **3D**: Three.js / React Three Fiber (Ambient only)
- **Deployment**: Node.js server (Next.js)

## Structure
- `/backend`: Django project root
- `/frontend`: Next.js project root

## Core Principles
- **Content First**: Server-rendered/Static critical content.
- **Performance**: Lighthouse >= 90.
- **Accessibility**: Reduced motion respect, semantic HTML.
